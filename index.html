Math DragOn

The tool to help you with your math homework

Download now

Product

More understandable

Because you are dragging the formulas together yourself, you will get a better understanding of how the formula works. Furthermore it will be a lot easier to maintain an overview of the formula because it is not just one big line of numbers and parameters.

Easier to use

With the use of a drag and drop interface the app is able to do copmplex mathematical calculations. It can both give approximations and Symbolic solutions. With this app you can solve problems far more easy then you are used to with a graphical calculator.

Symbolic solutions

As said, the app can solve problems symbolically. So when you are looking for derrivatives or integrals this app should be able to help you with a symbolic answer. The answers are not limited to constant values, but can also be formulas (e.g. when calculating the derrivative of x2 it will give 2x as an answer).

Features

Create your own functions

With secondary school maths there is no function too strange or too big for the app, so it will be able to help you to solve exercise you need help with.

Drag and drop interface

Easily build complex expressions using our easy to use drag and drop interface.

solve expressions symbolically or approximate them

It does not matter whether the exercise is to solve something symbolically or calculate it, the app is capable of doing both. It can solve any problem the way you want it to be solved.

Open and save functions

If there is a function of which you know you are going to need a lot, you can bookmark it for later user..
